# ŻabHop

Mobilny radar prowadzący do najbliższej Żabki albo — po wybraniu jednego przełącznika — do najbliższego sklepu innej popularnej sieci. Działa jako PWA: korzysta z lokalizacji i kompasu telefonu, pokazuje pięć najbliższych celów i otwiera pieszą trasę w Apple Maps.

## Uruchomienie

Otwórz stronę w Safari, stuknij **Znajdź najbliższą** i zezwól na lokalizację oraz kompas. Opcja **Dodaj do ekranu początkowego** instaluje wersję PWA.

Katalog Żabek jest lokalny. Dane innych sieci: © OpenStreetMap contributors, ODbL; lokalny snapshot ma zapasowe wyszukiwanie Photon/Overpass.
