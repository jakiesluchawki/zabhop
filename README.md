# ŻabHop

Mobilny radar prowadzący do najbliższej Żabki. Działa bez instalacji: korzysta z lokalizacji i kompasu telefonu, pokazuje pięć najbliższych sklepów i otwiera pieszą trasę w Apple Maps.

## Uruchomienie

Otwórz stronę w Safari, stuknij **Odpal radar** i zezwól na lokalizację oraz kompas.

Dane sklepów pochodzą z OpenStreetMap przez Photon, z zapasowym wyszukiwaniem Overpass.
